Nama Proyek: On The Road Website
On The Road Website .Build with boostrap, css, html, JavaScript, jquery, etc.
Deskripsi: 
Proyek ini adalah sebuah website yang dirancang untuk perusahaan Club motor yang Bernama On THe Road.
Website ini menggunakan Bootstrap untuk desain responsif, PHP untuk menangani form dan menghubungkan ke database MySQL,
serta memiliki beberapa bagian seperti, about, produk, dan kontak.

Fitur Pengguna:
1. Navbar
2. Landing Page Profil
3. Penawaran Produk
4. Link Vidio Promosi Social Media
5. About Us / Tentang Kami
6. Visi dan Misi
7. galeri
8. Footer

Fitur Admin:
1. Login Page
2. Salam dan Nama admin
3. Jam realtime
4. Create New Admin Account
5. Edit Account
6. Tambah Anggota
7. Logout
